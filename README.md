## Project Overview
This project was developed for **Olist**, a Brazilian marketplace, to centralize fragmented business insights into a single interactive dashboard.

As a **BI Analyst**, I designed a Tableau dashboard to monitor **revenue performance, customer behavior, and delivery operations**, enabling management and operations teams to make faster and more data-driven decisions.

## Objectives  
- Monitor overall business performance (e.g. revenue, orders)  
- Analyze customer growth, behavior, and retention  
- Evaluate delivery performance and operational efficiency  

## Dashboard Features  
- KPI Cards: Revenue, Orders, Customers, Delivery Metrics  
- Trends: Revenue, Orders, Customer Growth, Delivery Time  
- Customer Analysis: Segmentation, Frequency, Cohort
- Operational Insights: On-time vs Late Delivery, Delivery Type  
- Geographic Analysis: Revenue & Delivery Performance by State  
- Filters: Year-based dynamic analysis
